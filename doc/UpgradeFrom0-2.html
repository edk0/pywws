Python software for USB Wireless WeatherStations

UpgradeFrom0-2.py -- converts v0.2 datastore to v0.3

This program converts weather data from the format used by v0.2 of the Python weather station software to that used by v0.3. It is a good idea to back up your data before running UpgradeFrom0-2.py.

The program is very simple to use: 

python UpgradeFrom0-2.py data_directory

where data_directory is the location of your stored data.
