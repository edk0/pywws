Python software for USB Wireless WeatherStations

AutoDoc.py -- creates detailed documentation of each module

This program uses pydoc to create detailed documentation for each module / program, using the "docstrings" every Python file should contain. It should be run whenever you upgrade to a newer version of pywws.

The program is very simple to use: 

python AutoDoc.py
